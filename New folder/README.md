# NGO Accounting Mobile App

A Flutter mobile app for NGO accounting with transaction management, budget tracking, project reports, and user authentication.

## Features

- User authentication with email/password sign in
- Transaction dashboard with totals and recent entries
- Budget tracking with progress indicators
- Project report summaries and progress views
- Simple, responsive UI using Provider state management

## Setup

1. Install Flutter: https://flutter.dev/docs/get-started/install
2. Open this folder in your terminal
3. Run `flutter pub get`
4. Run `flutter run`

## Structure

- `lib/main.dart` — application entrypoint and state model
- `lib/screens/` — UI pages
- `lib/models/` — data model classes
- `lib/services/` — authentication and seeding helpers
- `lib/widgets/` — reusable UI components
